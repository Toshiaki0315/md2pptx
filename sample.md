## Mermaid図形の自動生成テスト
テキストの後に ````mermaid ```` を書くと、自動的に画像化されて右側に配置されます。

* **API**: Kroki.ioを利用
* **処理**: テキスト -> Base64圧縮 -> PNG取得
* 複雑な図解もテキストエディタだけで完結します！
```mermaid
graph TD
    A[ユーザーのMarkdown] -->|md2pptx.py| B(HTML解析)
    B --> C{タグ判定}
    C -->|img/table| D[通常レイアウト]
    C -->|mermaid| E[Kroki API呼び出し]
    E -->|PNG画像| F[スライドへ挿入]
```
